# Карточки вопросов (Streamlit)

Мобильное веб‑приложение: «тяните карту» со случайным вопросом и задавайте его партнёру/другу.

## Изменения в версии 1.2
- Починил переключение светлой/тёмной темы: меняется сразу
- Убраны лайки/избранное и вкладка избранного (осталась история)
- Убрано предупреждение про 18+
- Добавлена тема **Микс**: вопросы из всех разделов вперемешку
- Символ внизу карты зависит от категории (у «Романтика» — сердечко)

## Запуск
```bash
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -r requirements.txt
streamlit run app.py
```
